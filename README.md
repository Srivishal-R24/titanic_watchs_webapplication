# titanic_watchs_webapplication
A modern and responsive eCommerce watch website built using HTML, CSS, and JavaScript. This project showcases a stylish UI with smooth animations, product sections, and interactive features like cart, sliders, and particle background effects.
🚀 Features
🏠 Home Section with attractive hero design
⭐ Featured Products section
🛍️ Product Listing with pricing
🆕 New Arrivals Slider (Swiper.js)
🛒 Shopping Cart UI (add/remove items)
🌙 Dark/Light Theme Toggle
🎯 Smooth Scroll Navigation
💬 Testimonials Section
📩 Newsletter Subscription Form
✨ Particles.js Animated Background
📱 Fully Responsive Design
